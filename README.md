当前购物车逻辑：
 **保留 `cart` 表中的数据，只删除 `cart_items`**

- 这样可以保留购物车的存在，即用户的购物车始终存在，只是里面的商品被清空了。
- 当用户再次添加商品时，不需要重新创建购物车。
- 适用于 **“每个用户只有一个购物车”** 的逻辑。

即使 `cart_items` 被清空了，但 `cart` 仍然保留了用户的购物车记录。
